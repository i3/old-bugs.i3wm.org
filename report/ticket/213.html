Set Background Color

Reported by: rent0n Owned by: michael

Cc: Is it reproducible?: Always
git version: Operating system: Affects all

Your IRC nick:

Description

When using i3 with urxvt, urxvt always appears with some black borders around.

See screenshot: ​http://upload.centerzone.it/viewer.php?file=90295780956945721601.png

This is due to an urxvt bug that is not going to be fixed anytime soon.

By allowing the user to select i3 background color (as in wmii) the graphical issue would be solved as the user could set the same color for i3 and urxvt backgrounds.

Please add this feature in the next i3 releases.
Thank you.
