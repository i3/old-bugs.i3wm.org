This is actually working as intended.

When you select a stacked/tabbed container (by using focus parent), the X11 input focus cannot be set on any specific client (after all, you selected all of them). Also, focus down/focus up probably does not work because you donâ€™t actually have a container below the stacked container. focus left/right after focus parent works very well here (with two stacked containers next to each other).
