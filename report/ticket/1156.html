Ubuntu 12.04 and earlier has yajl version < 2. There are other required libraries you are most likely missing as well.

The docs say we support version 1.0.8.

comment:2 Changed 15 months ago by michael

I looked at this and it seems like yajl < 2 does not support JSON comments, which we generate in i3-save-tree(1).

Therefore, the fix for this is to raise the minimum yajl dependency, since we cannot make it compatible with both major versions.

I’ll do this during next week to give you guys some time to prove me wrong ;).
