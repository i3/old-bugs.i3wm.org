Description

The i3-dmenu-desktop perl script uses File::Temp::tempfile to generate a filename for a temporary file when 'Terminal' is true in a .desktop file (e.g. htop). A temporary shell script is created in the tempfolder and then executed. This does not work when the path to the temp folder contains spaces, because the exec is not adequately quoted.

Precondition:

export TMPDIR="/tmp/my silly tempdir"

Patch:

index 20c7fbe..16f0796 100755
--- a/i3-dmenu-desktop
+++ b/i3-dmenu-desktop
@@ -360,13 +360,13 @@ if (exists($app->{Terminal}) && $app->{Terminal}) {

binmode($fh, ':utf8');
say $fh <<EOT;

#!/bin/sh

-rm $filename
+rm "$filename"

exec $exec
EOT

close($fh);
chmod 0755, $filename;
