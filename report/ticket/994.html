Description

If I have multiple scratchpad windows open and try to toggle one (i.e. scratchpad show), the currently focused scratchpad window remains visible but the focus moves to another scratchpad window.

Previously, in 4.4, toggling the currently focused scratchpad window with scratchpad show would hide and move the focuse to another scratchpad window.

I have been able to reproduce this bug with my current patch : when multiple scratchpad are shown and you use "scratchpad show" without criteria, focus moves around windows.
