Crash when restart if I have rules for firefox

Reported by: rbgarga@… Owned by: michael
Priority: major Component: general
Version: 4.1 Keywords:
Cc: Is it reproducible?: Always
git version: Operating system: Linux
X11 program: Distribution / flavor: fedora 16 x86_64

Description

When I have the following lines in ~/.i3/config:

for_window [class="Firefox"] floating enable
for_window [class="Firefox" instance="Navigator"] floating disable

and firefox is running, if I try to restart i3, it crashes. I have
firefox 10.0.1 installed and i'm using the latest i3 code from "next"
branch (last commit == 45d5f1fbaa9a1d27ac22e33e5c4080d4e6fe9352).

When i comment these lines out, the problem is gone.

I collected a log and a core, and i'm attaching to these ticket.
