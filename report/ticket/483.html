comment:1 in reply to: ↑ description Changed 4 years ago by Merovius

Replying to elmico.filos@…:

It would be nice to have the option of launching a window menu listing all the clients, as it can be done in ion3 or awesome. Once open, users can select the client they want to go to using navigation keys (or the mouse).

imho this is possible using the ipc-interface already. You could query all managed clients per ipc-interface (get_tree messagetype), show a menu with something like dmenu (for example, or if you want something more complex) and focus the client via the X-Protocol.

Imo i3 should not get packed with too much of that stuff, since that is a rather specialized need and the ipc-interface was designed exactly for stuff like that.
